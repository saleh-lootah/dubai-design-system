# Input
The `dda-input` component is a customizable button element that can be used in various UI scenarios

### Usage in React

```jsx
import React from 'react'
import { DdaInput } from "@dubai-design-system/components-react";


const App = () => {
  return (
    <div>
        <DdaInput
            placeholder="Enter text"
            label="Text Input"
            value=""
            type="text"
            size="default"
            helper_text="Helper or footer text"
            error_message=""
            validation_type="normal"
            custom_class=""
            component_mode=""
            input_id="input"
            aria_label="input"
            onchange="console.log('changed')"
        ></DdaInput>
    </div>
 )
}

```
