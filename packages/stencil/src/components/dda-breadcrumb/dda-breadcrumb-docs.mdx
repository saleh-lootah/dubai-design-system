# Breadcrumb
The `dda-breadcrumb` component is a customizable button element that can be used in various UI scenarios

### Usage in React

```jsx
import React from 'react'
import { DdaBreadcrumb } from "@dubai-design-system/components-react";

const App = () => {
  return (
    <div>
        <DdaBreadcrumb
            design="text"
            separator="chevron-right"
            custom_class=""
            component_mode=""
            data-breadcrumbs='[{"text":"Home","icon":"home","url":"/"},{"text":"Library","icon":"menu_book","url":"/library"},{"text":"Data","icon":"storage","url":"/data"}]'
        ></DdaBreadcrumb>
    </div>
 )
}
```
