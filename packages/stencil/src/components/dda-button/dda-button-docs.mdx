# Button
The `dda-button` component is a customizable button element that can be used in various UI scenarios

### Usage in React

```jsx
import React from 'react'
import { DdaButton } from "@dubai-design-system/components-react";

const App = () => {
  return (
    <div>
        <DdaButton
            button_color="default-primary"
            start_icon="sentiment_satisfied"
            end_icon="arrow_forward"
            custom_class=""
            component_mode=""
            button_id="button"
            aria_label="button"
            onclick="console.log('clicked')"
            >Button
        </DdaButton>
    </div>
 )
}

```

### Usage in Vue

