# Number Field
The `dda-number-field` component is a customizable button element that can be used in various UI scenarios

### Usage in React

```jsx
import React from 'react'
import { DdaNumberField } from "@dubai-design-system/components-react";




const App = () => {
  return (
    <div>
        <DdaNumberField
            label="Amount"
            value=""
            placeholder="Enter amount"
            helper_text="Helper or footer text"
            error_message=""
            validation_type="normal"
            size="medium"
            input_status="normal"
            selected_currency="USD"
            custom_class=""
            component_mode=""
            input_id="input"
            aria_label="input"
            currencies='["USD","EUR","GBP","AED","PKR"]'
            onchange="console.log('changed')"
        ></DdaNumberField>
    </div>
 )
}

```
