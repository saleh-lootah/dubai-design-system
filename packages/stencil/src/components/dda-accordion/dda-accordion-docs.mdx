# Accordion
The `dda-accordion` component is a customizable button element that can be used in various UI scenarios

### Usage in React

```jsx
import React from 'react'
import { DdaAccordion } from "@dubai-design-system/components-react";


const App = () => {
  return (
    <div>
        <DdaAccordion 
        design="bg-border" 
        header_text="Accordion Header" 
        body_description="This is the accordion body content." 
        custom_class="" 
        component_mode="">
        </DdaAccordion>
    </div>
 )
}
```

### Usage in Vue

```jsx
<script>

import { DdaAccordion } from '@dubai-design-system/components-vue'

</script>

<template>
```
