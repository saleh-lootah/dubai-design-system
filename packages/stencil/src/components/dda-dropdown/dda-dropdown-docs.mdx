# Dropdown
The `dda-dropdown` component is a customizable button element that can be used in various UI scenarios

### Usage in React

```jsx
import React from 'react'
import { DdaDropdown } from "@dubai-design-system/components-react";


const App = () => {
  return (
    <div>
        <DdaDropdown
            label="Headline"
            selected=""
            disabled="false"
            error=""
            helper_text=""
            type="bg-white"
            size="default"
            icon_mode="false"
            custom_class=""
            component_mode=""
            button_id="dropdown"
            aria_label="dropdown"
            options='["Option 1","Option 2","Option 3"]'
        ></DdaDropdown>
    </div>
 )
}
```
