# Link Button
The `dda-link-button` component is a customizable button element that can be used in various UI scenarios

### Usage in React

```jsx
import React from 'react'
import { DdaLinkButton } from "@dubai-design-system/components-react";




const App = () => {
  return (
    <div>
        <DdaLinkButton
            button_color="default-primary"
            start_icon="sentiment_satisfied"
            end_icon="arrow_forward"
            custom_class=""
            href=""
            component_mode=""
            button_id="link-button"
            aria_label="link-Button"
            ><!---->Button
        </DdaLinkButton>
    </div>
 )
}
```
