# Footer
The `dda-footer` component is customizable that can be used in various UI scenarios

const App = () => {
  return (

    logo-alt="Dubai-Digital-Authority"
    sign-up-button-text="Sign Up"

    copyright-text="© 2023 Digitaldubai. All Rights Reserved."
