# Typography

This typography system is designed to ensure consistency across both the app and the website. It includes styles for both Arabic and English typography to ensure accessibility and localization. 

Below, you will find the different font styles used in the app and on the web, categorized by language.

### Accessibility Considerations
<br/>
The selected font sizes, line heights, and weights are designed to provide optimal readability for both Arabic and English languages. It is crucial to ensure that text is legible at all sizes, and that language-specific typographic needs (e.g., Arabic script readability) are respected.

This typography system standardizes the visual hierarchy for both App and Web environments, accommodating both Arabic and English users, ensuring consistency and usability across all platforms and languages.
