This document provides the standard effect styles used in web components to ensure a unified and accessible user experience. These effect styles are critical in enhancing the user interface by providing visual cues, depth, and focus to elements.

## Blur & Overlay

Blur and overlay effects are used for modals, pop-ups, and image backgrounds, adding a layer of depth and focus while softening the elements behind the overlay.

## Accessibility Considerations

Shadows should not interfere with readability or contrast. Keep shadows subtle enough to create depth without overwhelming the user's attention.
Focus styles must be clear and prominent to ensure keyboard navigation users can easily identify where their focus is.
Blur effects should not overly obstruct background content unless intentionally required. Ensure that text and other important elements remain readable.
