# Iconography
<br/>
This iconography system is designed to ensure a consistent usage of icons across web components. The system supports a variety of icon libraries for different use cases, from branding to functionality, and includes custom icon sets such as avatars and monograms.

Each icon set serves a specific purpose within the interface, ensuring clarity and enhancing user experience.
<br/>

### Usage in components

Material icons are used for visual emphasis, where icons need to stand out more, offering a combination of two colors within the same icon for a unique look.

```bash
<i class="material-icons  material-symbols-outlined">home</i>
```

### Accessibility Considerations

This iconography system ensures a cohesive design by leveraging a variety of icon libraries and custom sets. By following these guidelines, you can maintain consistency, clarity, and accessibility across all web components.
